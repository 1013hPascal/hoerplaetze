# Hörplätze -- Stadion Sound Finder

Dieses Repository sammelt Informationen zu Audio- und Hörplatz-Angeboten
für blinde und sehbehinderte Fußballfans in Deutschland.

Der Fokus liegt auf den Vereinen der: - 1. Bundesliga - 2. Bundesliga -
3. Bundesliga

## Aufbau

Die Daten sind bewusst einfach gehalten und liegen als Textdateien vor.

Struktur:

Leagen/ 1.Bundesliga/ 2.Bundesliga/ 3.Bundesliga/

Jede Textdatei steht für einen Verein und enthält alle relevanten
Informationen zum Hören eines Spiels im Stadion.

## Ziel

Die Inhalte aus diesem Repository werden von einer Web-App eingelesen,
sodass man schnell herausfinden kann: - ob Audiodeskription angeboten
wird - wie der Empfang funktioniert (z. B. WLAN, UKW, Funk) - wo man
Geräte bekommt - an wen man sich wenden kann

## Mitmachen

Falls Informationen fehlen oder nicht mehr aktuell sind, können diese
ergänzt werden.

## Hinweis

Die Informationen stammen aus unterschiedlichen Quellen und können sich
ändern. Im Zweifel empfiehlt es sich, direkt beim jeweiligen Verein
nachzufragen.
